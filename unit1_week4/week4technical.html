The group_by method is part of the enumerable class in ruby. It takes an array and converts it to a hash that is sorted by the input of the method. Here are some uses:
 You could sort an array of strings by first letter:

 strings = ["Lorem", "Ipsum", "Dolor", "Sit", "Lamet"]
 strings.group_by { |x| x[0] } => {"L"=>["Lorem", "Lamet"], "I"=>["Ipsum"], "D"=>["Dolor"], "S"=>["Sit"]}

 As seen above, group_by can be used to sort the list into a hash where the keys are letters and the values are the strings that begin with those letters. This is extremely useful for any sort of filing system, and is very easy to implement. Here's an example where the array is separated into a hash by length:

 strings.group_by { |x| x.length } => {"5"=>["Lorem", "Ipsum", "Dolor", "Lamet"], "3"=>["Sit"]}

 As you can see the strings array was converted into a hash with the keys being the length of the string and the values being the strings themselves. This is just the tip of the iceberg when it comes to the group_by method, but from these examples you can get a taste of the utility of the method.
